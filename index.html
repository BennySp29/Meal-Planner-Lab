☁️ Real-time Sync Setup

Connect a free Firebase Realtime Database so you and your partner see updates instantly.

1
Go to console.firebase.google.com and create a free project
2
Click Build → Realtime Database → Create database and choose "Start in test mode"
3
Go to Project Settings (⚙️) → General → Your apps → Add web app and copy the config values below
4
Share this HTML file with your partner — you'll both enter the same Firebase config to sync
